
Frontend Quickstart
-------------------
1) Lokal serven (z.B.):
   - Python:  python -m http.server 8000
   - Node:    npx serve
2) Wenn Backend auf anderem Host/Port läuft, setze vor dem Laden irgendwo:
   <script>window.API_BASE='http://localhost:8080';</script>
3) Öffne http://localhost:8000/index.html
